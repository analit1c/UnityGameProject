Educational Concept
The pedagogical goal of this game is to improve the player’s spatial awareness, reaction speed, strategic thinking, and problem-solving abilities through interactive gameplay challenges.

The project mainly applies the Constructivism learning theory. Players learn through direct interaction, experimentation, and experience inside the game world. Instead of receiving information passively, the player actively develops skills by overcoming obstacles, navigating mazes, and adapting to enemy behavior.

The game mechanics facilitate learning in several ways. Maze exploration improves memory and navigation skills, while parkour sections develop timing and coordination. Combat against robots encourages quick decision-making and strategic thinking under pressure. Players continuously analyze the environment, react to threats, and improve their performance through repeated gameplay experiences.
